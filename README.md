# 결정적 디버깅? (Deterministic debugging?)
WDD is record/replay based deterministic debugger.  
Implements [rr](http://rr-project.org/) ideas on the windows platfrom.

# How to use

## To record

```
wdd record yourprogram.exe
```

## To replay

```
wdd replay yourprogram.exe
```
